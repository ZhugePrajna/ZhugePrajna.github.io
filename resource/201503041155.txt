用如下方法验证:

curl --ciphers DEFAULT --compressed -ksL -I ...
curl --socks5 <ip>:<port> --ciphers DEFAULT --compressed -ksL -I ...
curl --proxy socks5h://<ip>:<port> --ciphers DEFAULT --compressed -ksL -I ...

socks5h是指远程DNS解析，新版curl支持
